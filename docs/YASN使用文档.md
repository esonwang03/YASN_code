#### 便签的三种层次模式
- Normal：正常模式。在此模式下，便签的层次会按照当前软件窗口的层次排布。比如当前便签在第四层，那么在正常模式下会保持在第四层。
- Top Most：置顶模式。在此模式下，无论原先便签在第几层，它都会保持在顶层。无论有什么其他新窗口打开，它都会保持在最顶层。
- Bottom：置底模式。在此模式下，无论原先便签在第几层，它都会保持在最底层，只有在桌面才能看到。

>注意：
>置顶模式和置底模式支持多个窗口共同设置。但它们也会有层次关系，只是相比其他软件置于最底了。
>在点击“显示桌面”的时候，便签也会隐藏。在点击其他软件的时候便签会重新出现。

### 编辑便签
#### 便签的三种编辑模式
- 双栏模式，此时在左侧可以编辑markdown内容，右侧会实时渲染效果。
- 阅读模式，此时仅展示渲染效果，适合在编辑完成后进行展示。
- 源码模式，此时仅展示markdown源码内容。
#### 编辑过程
首次打开便签，默认进入双栏模式，此时可以使用markdown语法在左侧编辑便签内容，在右侧可以实时渲染。

![[Pasted image 20260627161830.png]]

编辑完成后，保证光标聚焦在左侧的markdown文本框中，此时**按下`Esc`键**即可切换为阅读模式。
此时我们如果想编辑内容，**双击要编辑的位置**，即可切换为双栏模式进行编辑。双栏模式和阅读模式是最常用的两个模式。
当然，点击右上角的第一个按钮，即可切换为各种模式。

![[Pasted image 20260627163339.png]]

## 进阶功能
### Webdav同步
如果想要在多台设备中同步便签内容，则需要配置一个**支持Webdav的网盘服务器**。这里以[坚果云](https://www.jianguoyun.com/)为例。
#### 坚果云配置
安装坚果云客户端后，在左下角设置中点击“第三方应用管理”。

![[Pasted image 20260627171319.png]]

在第三方应用管理中，记住服务器地址和账号，然后点击添加应用密码。
![[Pasted image 20260627171417.png]]

输入名称后，点击生成密码。

![[Pasted image 20260627171602.png]]

点击完成并复制。

![[Pasted image 20260627171634.png]]

随后我们在坚果云新建一个同步文件夹，用于保存要同步的便签内容。具体操作按照坚果云的提示即可，假设我们的同步文件夹名称为`YASN_data`。

![[Pasted image 20260627172421.png]]

此时完成了坚果云的配置。我们后续在第三方应用管理中还可以复制这个密码。
#### YASN配置
进入YASN的设置界面，在同步这里点击启用同步功能，此时填写`serverURL`（服务器地址），`username`（账号）（这两个字段在上面第三方应用管理界面有展示），密码是刚刚复制的应用密码；`remote directory`填刚刚创建的同步文件夹名称，我这里是`YASN_data`；其它设置可保持默认，点击测试连接，通过后保存即可。

![[Pasted image 20260627172238.png]]

关闭设置界面后，如果需要将服务器上的便签内容同步到本机，还需要点击`Sync now`按钮手动完成一次同步，此时再刷新一下即可显示同步内容。

![[Pasted image 20260627172925.png]]

### 自定义便签样式
YASN支持自定义便签样式展示。需要自己写好CSS文件，随后点击笔记管理界面中点击`Open data folder`，进入`style`文件夹，将css文件复制在其中。

![[Pasted image 20260627173627.png]]

当然也可以复制在`style`文件夹下的二级目录中。如图所示，我们将css文件保存在katex文件夹中，这样在设置同样会显示。

![[Pasted image 20260627173746.png]]

这样在设置的preview style选项这里就可以看到自定义的样式文件名，选中保存即可切换。
